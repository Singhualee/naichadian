# 🧋 奶茶店经营模拟游戏

一款基于浏览器的奶茶店经营模拟游戏，玩家需要管理原料采购、定价、雇员招聘等，制定经营策略实现盈利。

## 🎮 游戏玩法

### 核心循环
1. **采购原料** - 根据销售需求购买牛奶、茶叶、糖浆等原料
2. **制作产品** - 制作各种奶茶产品出售
3. **定价策略** - 调整产品价格平衡利润与销量
4. **招聘雇员** - 雇佣员工提升服务效率和客流量
5. **结束营业** - 每天结束后查看经营报表，分析经营数据

### 产品配方
| 产品 | 原料配方 | 基础售价 |
|------|---------|---------|
| 原味奶茶 | 牛奶×1, 茶叶×1, 糖浆×1 | ¥12 |
| 波霸奶茶 | 牛奶×1, 茶叶×1, 糖浆×1, 珍珠×1 | ¥15 |
| 水果茶 | 水果×2, 茶叶×1, 糖浆×1 | ¥18 |
| 抹茶拿铁 | 抹茶粉×1, 牛奶×1, 糖浆×1 | ¥22 |

### 原料系统
- 原料价格会根据市场行情波动
- 不同原料有不同保质期，过期将损失
- 批量采购可能获得更优惠的价格

### 成本计算
- 产品成本按实际原料消耗计算
- 每日固定成本：房租 + 水电 + 员工工资（按天分摊）
- 合理定价是盈利的关键

## 📊 经营分析功能

游戏内置强大的经营分析系统，帮助玩家制定决策：

### 经营概况
- 平均日收入/成本/利润
- 客单价分析

### 雇员影响分析
- 评估增加雇员带来的顾客增量
- 工资成本 vs 收益分析

### 定价影响分析
- 涨价对销量的预估影响
- 价格弹性模拟

### 原料涨价影响
- 原料价格上涨对利润的影响预估
- 风险评估

### 盈亏平衡分析
- 计算保本点
- 制定经营目标

## 🚀 如何开始

### 在线运行
1. 克隆或下载本项目
2. 使用浏览器直接打开 `index.html`
3. 或使用本地服务器运行：
   ```bash
   python -m http.server 8080
   ```
4. 访问 `http://localhost:8080`

### 游戏难度
- **简单模式** - 初始资金 ¥50,000
- **普通模式** - 初始资金 ¥30,000
- **困难模式** - 初始资金 ¥10,000

## 💾 存档功能

- 支持3个存档位
- 支持自动保存
- 数据保存在浏览器本地存储

## 🎯 经营提示

1. **原料管理**：注意原料保质期，避免浪费
2. **定价策略**：价格太高会流失客户，太低则利润不足
3. **雇员配置**：雇员能带来更多顾客，但需要支付工资
4. **市场波动**：关注原料价格变化，低价时适当囤货
5. **数据分析**：经常查看经营分析，做出数据驱动的决策
